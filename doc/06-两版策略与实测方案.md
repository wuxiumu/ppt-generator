## 06 - 两版策略与实测方案

### 两版的核心差异

| 维度 | 美观极致版 | 平衡版 |
|------|-----------|--------|
| 核心目标 | 在$10/30分钟内做到最好看 | 在$1/5分钟内做到够用 |
| Planner模型 | Claude 3.5 Sonnet | GPT-4o-mini |
| Writer模型 | Claude 3.5 Sonnet | GPT-4o-mini |
| 图片来源 | DALL-E 3 HD生成 | Unsplash免费图库 |
| 图表样式 | 精调matplotlib + 设计感 | 基础matplotlib样式 |
| 布局复杂度 | 全12种布局 + 装饰元素 | 8种核心布局 |
| 后处理 | 叙事校验 + 视觉微调 | 仅叙事校验 |
| 并行度 | 5（API限流考量） | 10（速度优先） |

### 美观极致版的"加分操作"

在成本允许范围内（预算$10，实际LLM成本<$2），极致版可以额外做：

**1. 多次规划择优（+$0.16）**
调用2次Planner，用不同的叙事策略，然后让一个Judge LLM选择更好的方案。

```python
async def plan_with_judge(planner, input_data):
    # 两种不同策略
    plan_a = await planner.plan(input_data, strategy="storytelling")
    plan_b = await planner.plan(input_data, strategy="analytical")
    
    # Judge评估
    judge_prompt = f"""
    比较以下两个PPT规划方案，哪个更好？
    评估维度：叙事连贯性、受众适配度、内容丰富度、布局多样性
    
    方案A: {plan_a.to_json()}
    方案B: {plan_b.to_json()}
    
    输入信息: {input_data}
    """
    verdict = await call_llm("gpt-4o", judge_prompt)
    return plan_a if verdict.prefers == "A" else plan_b
```

**2. 内容精修轮次（+$0.02）**
Writer生成初稿后，调用一次"编辑"LLM做精炼：

```python
async def polish_content(draft, slide_spec):
    polish_prompt = f"""
    精炼以下PPT slide的内容。要求：
    - 文字减少20-30%，保留核心信息
    - 让表述更有冲击力
    - 保持语气一致
    
    初稿: {draft.to_json()}
    """
    return await call_llm("claude-3-5-sonnet", polish_prompt)
```

### 平衡版的"省钱操作"

**1. 降低Planner模型等级**
GPT-4o-mini 的规划能力弱于Sonnet，所以：
- 提供更详细的prompt模板（把决策空间缩小）
- 预设更多叙事模板，Planner只需"选择+微调"而非"从零规划"

```python
# 平衡版的Planner不是自由规划，而是从模板库中选择最接近的
BALANCED_PLANNER_PROMPT = """
从以下5个叙事模板中选择最适合当前主题的：
1. 渐进教程型：封面→问题→概念A→示例→概念B→示例→练习→总结→下一步
2. 时间线型：封面→起点→里程碑1→里程碑2→里程碑3→反思→展望
3. 决策指南型：封面→需求分析→维度1→维度2→维度3→决策框架→推荐
4. 说服提案型：封面→痛点→现状分析→方案→证据→ROI→行动
5. 行程攻略型：封面→总览→Day1上午→Day1下午→Day2上午→Day2下午→Tips

选择模板后，只需填充每张slide的content_brief，不要改变结构。
"""
```

**2. Unsplash 代替 DALL-E**

```python
async def search_unsplash(query: str, palette_name: str) -> str:
    """搜索与主题和配色匹配的免费图片"""
    async with httpx.AsyncClient() as client:
        resp = await client.get(
            "https://api.unsplash.com/search/photos",
            params={
                "query": query,
                "per_page": 5,
                "orientation": "landscape"
            },
            headers={"Authorization": f"Client-ID {UNSPLASH_KEY}"}
        )
        photos = resp.json()["results"]
        # 选择最匹配的一张
        return photos[0]["urls"]["regular"]
```

### 踩坑预案

| 可能遇到的问题 | 预案 |
|--------------|------|
| Claude API 限流 | 备用 GPT-4o，自动切换 |
| DALL-E 3 生成太慢 | 降级为 DALL-E 3 标准质量（非HD） |
| python-pptx 渲染异常 | 备用方案：生成HTML再用LibreOffice转pptx |
| 中文字体缺失 | 内嵌字体文件，或用Arial替代 |
| 叙事校验太严格 | 设置severity等级，只阻断critical问题 |
| 单张slide内容溢出 | 自动拆分为两张，保持布局不变 |
