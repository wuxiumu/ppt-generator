## 03 - 视觉风格系统：设计系统 + 布局模板 + 配色方案

### 设计哲学

PPT不是文档，是"视觉化的演讲辅助"。设计原则：

**留白优先**：每张slide只用60-70%的面积放内容，其余留白
**信息层级**：标题 > 关键数据/图 > 正文 > 注释，视觉权重递减
**一页一重点**：每张slide只传达一个核心信息
**呼吸感**：字号、间距、对齐严格统一，让受众感觉"舒服"

### 布局模板库（12种）

```
L01 - Title        封面页（大标题+副标题+装饰元素）
L02 - Section      章节分隔页（大色块+章节名+序号）
L03 - Statement    声明页（一句话居中，大字）
L04 - Bullets      要点列表页（3-5个要点+图标）
L05 - Image+Text   图文混排（左图右文 或 上文下图）
L06 - FullImage    全图页（图片铺满+半透明文字叠加）
L07 - Comparison   对比页（左右两栏对比）
L08 - Timeline     时间线页（横向或纵向时间轴）
L09 - BigNumber    大数字页（关键数据突出显示）
L10 - Chart        图表页（柱状图/饼图/折线图）
L11 - CodeBlock    代码展示页（代码高亮+注释）
L12 - End          结束页（感谢+联系方式）
```

### 视觉节奏控制

25-30张PPT不能全是同一种布局。需要控制视觉节奏：

```
推荐的视觉节奏模式（30张）：

S01  Title        ████████████████████  封面，视觉冲击力强
S02  Statement    ████████░░░░░░░░░░░░  钩子，文字聚焦
S03  Bullets      ████████████░░░░░░░░  共鸣要点
S04  Timeline     ██████████████░░░░░░  路线图
S05  Section      ████████████████████  章节分隔，色块切换
S06  Image+Text   ██████████████░░░░░░  图文并茂
S07  Bullets      ████████████░░░░░░░░  
S08  CodeBlock    ██████████████░░░░░░  代码（如果是技术主题）
S09  BigNumber    ████████████░░░░░░░░  数据冲击
S10  Section      ████████████████████  章节分隔
...（重复主体模式，每3-4张变一次布局）
S25  Comparison   ██████████████░░░░░░  对比总结
S26  Statement    ████████░░░░░░░░░░░░  金句
S27  Bullets      ████████████░░░░░░░░  行动号召
S28  End          ████████████████████  结束，首尾呼应
```

**关键规则**：
- 连续2张相同布局是上限，第3张必须换
- Section分隔页是视觉呼吸点，间距不超过6张
- BigNumber 和 Statement 是"节奏变化器"，穿插使用
- 封面和结束页使用相同配色但不同布局，形成"回环"

### 图表生成策略

图表不使用python-pptx原生图表（样式难控），而是用 matplotlib 生成图片后嵌入：

```python
import matplotlib.pyplot as plt

def create_chart(chart_data, palette):
    plt.style.use('seaborn-v0_8-whitegrid')
    fig, ax = plt.subplots(figsize=(8, 5), dpi=150)
    
    # 应用调色板配色
    colors = [palette['accent_1'], palette['accent_2'], 
              palette['accent_3'], palette['accent_4']]
    
    if chart_data['type'] == 'bar':
        ax.bar(chart_data['labels'], chart_data['values'], color=colors)
    elif chart_data['type'] == 'pie':
        ax.pie(chart_data['values'], labels=chart_data['labels'], colors=colors)
    elif chart_data['type'] == 'line':
        ax.plot(chart_data['labels'], chart_data['values'], color=colors[0])
    
    # 统一样式：无边框、圆角、大字号
    ax.spines['top'].set_visible(False)
    ax.spines['right'].set_visible(False)
    ax.tick_params(labelsize=12)
    
    fig.savefig(temp_path, bbox_inches='tight', transparent=True)
    return temp_path
```
